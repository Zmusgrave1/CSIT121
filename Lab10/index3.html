How XML Base (2nd Edition) is Used

XML Base (2nd Edition) is a specification by the World Wide Web Consortium (W3C) that defines a facility for defining base URIs for parts of XML documents. This is similar to the HTML BASE element, which allows authors to specify a base URI for resolving relative URIs within a document.

Usage of XML Base

To use XML Base, you can include the xml:base attribute in your XML elements. This attribute specifies a base URI for resolving relative URIs within the scope of the element it is applied to.

Example 1: Basic Usage

Here is a simple example of how to use the xml:base attribute:

        
            <root xml:base="http://example.com/base">
                <element>Content</element>
                <link href="relative/path/to/resource"/>
            </root>

In this example, the relative URI relative/path/to/resource will be resolved against the base URI http://example.com/base.

Example 2: Nested Elements

XML Base can also be used with nested elements to define different base URIs for different parts of the document:

        
            <root xml:base="http://example.com/base">
                <element>Content</element>
                <section xml:base="http://example.com/section">
                    <link href="relative/path/to/section/resource"/>
                </section>
            </root>

In this example, the relative URI relative/path/to/section/resource will be resolved against the base URI http://example.com/section, while other relative URIs in the root element will be resolved against http://example.com/base.

Example 3: Changing Base URIs

You can change the base URI within the same document by applying the xml:base attribute to different elements:

        
            <root xml:base="http://example.com/base">
                <element>Content</element>
                <section xml:base="http://example.com/section">
                    <link href="relative/path/to/section/resource"/>
                </section>
                <footer xml:base="http://example.com/footer">
                    <link href="relative/path/to/footer/resource"/>
                </footer>
            </root>

In this example, the relative URI relative/path/to/footer/resource will be resolved against the base URI http://example.com/footer.

Conclusion

XML Base (2nd Edition) is a powerful tool for managing base URIs in XML documents. By using the xml:base attribute, authors can ensure that relative URIs are resolved correctly, improving the overall consistency and reliability of XML documents.
